# msgvc6w
msg.exe (without console, built on VC++6)

A Windows app that calls MessageBoxW Windows API.

Usage
-----

msg.exe "Hello World!" "Message"

msg.exe * "Hello World!" "Message"

msg.exe "Hello World!"
